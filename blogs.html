What is DOM ? Write its Purpose

DOM means Document Object Model. (DOM) is a programming interface for web documents. It represents the page so that programs can change the document's structure, style, and content.

How can we select element from Dom?

We can select DOM elements in five ways

  • querySelectorAll()
  • querySelector()
  • getElementsByClassName()
  • getElementById()
  • getElementsByTagName()

    • What is callback function?

    A callback function is a function that is passed as an argument to another function, then called inside the outer function to complete action.
